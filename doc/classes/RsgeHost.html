Class RsgeHost
In: lib/rsgehost.rb
Parent: Object

RsgeHost has methods for obtaining information about hosts in a GridEngine environment. Eventually, we will also include methods for setting host complex values, enabling/disabling, add/remove, hostgroup associations, etc.
